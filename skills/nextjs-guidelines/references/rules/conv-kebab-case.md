## Use kebab-case for files and directories

All file and directory names use kebab-case. Component file names match the component name in kebab-case.

**Incorrect:**

```
transactionList.tsx
TransactionList.tsx
use_transaction_form.ts
transaction_list.tsx
```

**Correct:**

```
transaction-list.tsx
use-transaction-form.ts
transaction-service.ts
app-sidebar.tsx
```
